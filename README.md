# IPL-Player-Performance-Analytics-Platform-with-KPI-Dashboard
Python-based IPL analytics platform using Pandas/NumPy. It cleans data to compute KPIs like strike rates and averages. Features a modular SQL/CSV backend and Streamlit dashboard for trend tracking.
This project presents an IPL Player Performance Analytics Platform with KPI Dashboard, designed to enable data‑driven assessment of cricketers’ performance across multiple IPL seasons. Built in Python using Pandas and NumPy, the platform ingests historical IPL match and player‑stat datasets, performs comprehensive data cleaning, aggregation, and feature engineering, and computes key performance indicators such as runs scored, wickets taken, strike rate, economy rate, batting and bowling averages, and consistency metrics. These KPIs are stored in a structured format (CSV/SQL), enabling efficient querying, filtering, and comparative analysis at player, team, and season levels. An interactive dashboard, implemented using Streamlit or Dash, visualizes the metrics through dynamic line charts, bar plots, and heatmaps, allowing users to explore performance trends, form slumps, and season‑wise trajectories. The architecture follows a modular, layered design that separates data ingestion, transformation, storage, and presentation, improving maintainability and extensibility. The system is engineered to be adaptable to other T20 leagues and can support integration of advanced analytics such as impact‑index calculations, matchup‑based evaluations, and machine‑learning‑driven performance forecasts. Overall, this repository provides a production‑oriented, end‑to‑end analytics pipeline tailored to sports analytics, dashboard development, and the practical application of time‑series data processing in Python
